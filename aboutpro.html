Installation Steps:

  1. Navigate to the web folder:
    2. cd /var/www/html/
  2. Install git:
    3. sudo apt install git
  3. Associate your git username with this installation:
    4. git config --global user.name "[Name]"
  4. Associate your git email with this installation:
    5. git config --global user.email "Your email"
  5. Clone the repository into your web space:
    6. sudo git clone https://github.com/redpangilinan/dungeon-crawler-rpg-od.git

You should now be able to view the web page here: http://YourServerIP/dungeon-crawler-rpg-od/
